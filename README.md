## Como navegar

- As aulas estao organizadas em [`lectures/`](./lectures), uma pasta por tema.
- Cada pasta combina apresentações, scripts e, quando necessário, materiais auxiliares.
- Arquivos grandes de download local, como arquivos `.tar.gz` de datasets, não fazem parte do versionamento.
